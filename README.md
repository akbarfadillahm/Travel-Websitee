## ✨ Fitur

- ✅ **Navbar Responsive** — dengan efek transparan saat scroll
- ✅ **Hero Section** — dengan background full-screen dan search bar
- ✅ **Destinasi Populer** — grid layout asimetris (Bali, Lombok, Raja Ampat, dll)
- ✅ **Paket Tour** — card dengan harga dan tombol pesan
- ✅ **Stats Section** — angka statistik perusahaan
- ✅ **Keunggulan** — fitur unggulan layanan
- ✅ **Testimonial** — ulasan dari pelanggan
- ✅ **Newsletter** — form subscribe email
- ✅ **Footer** — lengkap dengan navigasi dan sosial media
- ✅ **Fully Responsive** — mobile, tablet, dan desktop

## 🎯 Bootstrap Components yang Dipakai

- `Navbar` + `Collapse` — navigasi responsive
- `Grid System` — layout kolom fleksibel
- `Card` — destinasi dan paket tour
- `Badge` — label dan tag
- `Form Control` & `Select` — search bar
- `Utility Classes` — spacing, flex, positioning

## 📖 Cara Menjalankan

1. Clone repository ini:
```bash
git clone https://github.com/akbarfadilahm/travel-website.git
```

2. Buka folder project:
```bash
cd travel-website
```

3. Buka `index.html` di browser — **selesai!** 🎉

> Tidak perlu install apapun karena Bootstrap menggunakan CDN.
